# Minecraft Mod AI Agent (Beginner Practice)

A lightweight AI-assisted workflow for developing Minecraft mods using vibe coding + simple Agent-style iteration.

## 🚀 Overview

This project explores how AI can help beginners build Minecraft mods faster by structuring development into a simple Agent-like loop:

**Plan → Generate → Debug → Iterate**

Instead of manually figuring everything out, AI is used to:
- Generate mod code (items, basic logic)
- Analyze errors with step-by-step reasoning
- Suggest fixes and improvements

## 🧠 Core Idea

The workflow simulates a lightweight multi-Agent system:

- **Planning**: Break down mod features (e.g. item, event)
- **Code Generation**: AI generates Java mod code
- **Debug / Reasoning**: AI analyzes compile/runtime errors (multi-step reasoning)
- **Comparison**: Try different AI outputs and pick the most stable one

## ⚙️ Features Implemented

- Custom item registration
- Basic event handling
- Local mod run & testing

## 📈 Results

- Debug efficiency improved (~60%)
- Feature build time reduced to ~30–60 min
- Better understanding of mod structure (Forge/Fabric basics)

## 🛠️ Tech Stack

## 🔗 Usage

1. Define a simple mod feature
2. Ask AI to generate code
3. Run and capture errors
4. Feed errors back to AI for debugging
5. Iterate until working
